# Intelligent Call Transcription & Behavioral Analytics

An AI-powered call center analytics platform that transforms customer call recordings into actionable business insights using **Automatic Speech Recognition (ASR)**, **Natural Language Processing (NLP)**, and **Large Language Models (LLMs)**.

## Features

* 🎙️ Automatic Speech-to-Text using Whisper ASR
* 😊 Sentiment Analysis
* 🎯 Intent Detection
* 📝 AI Call Summarization
* 🔍 Semantic Search with Vector Embeddings
* 💬 AI Chatbot for Natural Language Queries
* 🗄️ Text-to-SQL Query Generation
* 📊 Interactive Streamlit Dashboard
* 📈 Agent Performance & Customer Analytics

## Tech Stack

* Python
* Snowflake
* Snowflake Cortex AI
* Streamlit
* SQL
* Whisper ASR
* Vector Embeddings
* Semantic Search
* NLP & Large Language Models (LLMs)

## Workflow

1. Upload customer call recordings.
2. Convert audio into text using Whisper ASR.
3. Perform sentiment analysis, intent detection, and summarization with Snowflake Cortex.
4. Generate vector embeddings for semantic search.
5. Query transcripts using Chatbot or Text-to-SQL.
6. Visualize insights through an interactive Streamlit dashboard.

## Applications

* Call Center Analytics
* Customer Experience Management
* Agent Performance Evaluation
* Business Intelligence
* Customer Support Automation
